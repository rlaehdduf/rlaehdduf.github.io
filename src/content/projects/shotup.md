## 게임 개요

Unity 2D 로 개발한 탑다운 뷰 슈팅 게임입니다.

### 주요 기능

- **플레이어 컨트롤러**: WASD 이동 + 마우스 조준/발사
- **적 AI**: 플레이어 추적 및 공격 패턴
- **충돌 시스템**: 2D 물리 기반 충돌 처리
- **코어 구조**: Scripts/Config, Controllers, Core, Events, Services 분리

### 기술 스택

- Unity 2D 물리 시스템
- 이벤트 기반 아키텍처
- 서비스 로케이터 패턴
