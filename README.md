# My Portfolio

개인 프로젝트 포트폴리오 웹사이트

## 프로젝트 목록

- DefenceGame (Unity) - 타워 디펜스 게임
- GreedDungeon (Unity) - 로그라이트 던전 게임
- Pathfinder (Unity) - AI 경로 탐색 학습 프로젝트
- ShotUp (Unity) - 탑다운 슈팅 게임
- GameDevToolkit (Unity Package) - 게임 개발 툴킷
- TestWebPage (Node.js/Express) - 게시판 웹 애플리케이션
- TestWebText (Node.js) - 웹 텍스트 프로젝트
- WebMake (Node.js) - 웹 프로젝트 컬렉션
- Test0330 (Web) - 테스트 웹 프로젝트

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

GitHub Pages에 자동 배포됩니다.

### GitHub Pages 설정

1. GitHub 저장소 생성
2. Settings > Pages > Source: GitHub Actions
3. main 브랜치에 push 시 자동 배포

## 기술 스택

- Astro (정적 사이트 생성기)
- TypeScript
- CSS
- GitHub Pages (호스팅)
